# 安装 Mocha

安装 Mocha 比较容易。

## 1. 通过 npm  安装（推荐）

安装在全局：

```
npm install --global mocha

// 或者
// npm install -g mocha
```

安装在项目中：

```
npm install --save-dev mocha
```

> [版本要求](https://mochajs.org/#installation)： npm >= v1.4.0，Node.js >= v0.10。

## 2. 通过 Bower 安装

```
bower install mocha
```

## 3. cdnjs

详见 https://cdnjs.com/libraries/mocha 。
